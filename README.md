# Gabriel Barcelos — HTML Resume

A simple and clean **HTML resume** built with semantic tags and ready for customization.

## About
This project is a personal resume made using only **HTML5**.  
It’s lightweight, structured, and ready to be styled or converted into a portfolio page.

## Sections
- Header with name and contact info  
- Objectives  
- Education  
- Experience  
- Links (LinkedIn & GitHub)
